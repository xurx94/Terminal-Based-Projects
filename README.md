
# CLI Projects 

> A collection of my command-line interface (CLI) projects built over the years.

## Projects 

* **ObjectiveBankLtd (C++)**  
  A terminal-based banking system built using Object-Oriented Programming (OOP) concepts. Its main features include account creation, deposits, withdrawals, tracking transaction history, transferring funds, and paying utility bills.

## Setup

First, clone the repository to your local machine:

```bash
git clone
(https://github.com/xurx94/Terminal-Based-Projects.git)
cd Terminal-Based-Projects

```
## Usage
To run the ObjectiveBankLtd project, you will need a C++ compiler (like g++).
Compile the code:
```bash
g++ ObjectiveBankLtd.cpp -o bank_system

```
Run the compiled program:
```bash
# On Windows:
bank_system.exe

# On Linux/macOS:
./bank_system

```
